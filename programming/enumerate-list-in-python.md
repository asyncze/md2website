# Enumerate List in Python

*June 2018* [Python](programming.html#python) [Misc](programming.html#python-misc)

```python
my_list = ["Zero", "One", "Two", "Three", "Four", "Five"]

for item in enumerate(my_list): print(item)
# (0, 'First')
# (1, 'Second')
# (2, 'Third')
# (3, 'Fourth')
# (4, 'Fifth')
```
